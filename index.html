Por que Precisamos Entender os Impactos das Chuvas Intensas

Chuvas fortes e enchentes causam danos materiais, prejuízos econômicos e riscos à vida em áreas urbanas. Muitas comunidades enfrentam falta de informação e ferramentas simples para prever e se preparar para esses eventos.

Tecnologias que Tornam o EcoPluvius Possível

QGIS: Software de código aberto para tratamento de dados geográficos

Leaflet.js: Biblioteca JavaScript para mapas interativos e leves

HTML5, CSS3, JavaScript: Fundamentos para estrutura, estilo e interatividade

GeoJSON: Formato padrão para dados geográficos na web

GitHub Pages/Vercel: Hospedagem gratuita e rápida da aplicação

O Que Queremos Alcançar com o EcoPluvius

Como o EcoPluvius torna simulações geográficas acessíveis?
Através de uma interface interativa e amigável, o EcoPluvius permite que qualquer pessoa explore simulações de eventos naturais, como enchentes, sem necessidade de conhecimento técnico prévio.
Por que é importante visualizar riscos de enchentes?
Visualizações facilitam o entendimento de áreas de risco e impacto, ajudando na conscientização da população e na adoção de medidas preventivas de forma mais clara e rápida.
Quem pode se beneficiar dessas simulações?
Cidadãos, educadores, gestores públicos e pesquisadores podem usar o EcoPluvius como ferramenta de apoio para planejar ações, ensinar sobre mudanças climáticas ou analisar dados de eventos extremos.
Como o projeto apoia o desenvolvimento técnico?
O EcoPluvius serve como uma base sólida de dados e visualização, ideal para desenvolvedores criarem novas soluções, modelos de previsão, integração com sensores IoT e sistemas de alerta em tempo real.
